嗨！今天想和大家分享我做的第一个网页端小工具——**ChartGenius**。

工具的入口为：[ChartGenius](https://chart-genius-sooty.vercel.app/)

## 为什么会做这个工具？

做 ChartGenius 的想法其实源于一个很实际的需求：写实验报告和做课程展示时，我经常需要画各种图表。虽然市面上已经有很多画图工具了，但使用起来要么太复杂（学习成本高），要么美观度不够（配色看着就很"理工直男"），要么就是需要付费才能导出高质量图片。

我就在想，能不能做一个简单、快速、好看的图表生成器？最好是打开网页就能用，不需要安装任何软件，也不需要复杂的操作。于是，ChartGenius 就诞生了。

## ChartGenius 能做什么？

简单来说，ChartGenius 是一个**快速制作和美化图表的在线工具**。它的核心功能就是：

*   **快速制作图表**：几步操作就能生成专业的图表
*   **灵活美化**：一键切换配色方案，实时预览效果
*   **即开即用**：无需注册，打开网页就能用

## 功能详解

### 丰富的图表类型

ChartGenius 目前支持 **11 种**常用图表类型，基本能满足日常学习和工作中的大部分需求：

*   **柱状图 (Bar Chart)**：最经典的数据对比图表
*   **水平柱状图 (Horizontal Bar Chart)**：当标签文字较长时更适合
*   **堆叠柱状图 (Stacked Bar Chart)**：展示部分与整体的关系
*   **百分比堆叠柱状图 (Percentage Stacked Bar Chart)**：对比各部分的占比
*   **直方图 (Histogram)**：展示数据的分布情况
*   **折线图 (Line Chart)**：展示趋势变化
*   **面积图 (Area Chart)**：强调数值的累积
*   **饼图 (Pie Chart)**：展示各部分的占比关系
*   **环形图 (Donut Chart)**：饼图的变体，视觉上更现代
*   **雷达图 (Radar Chart)**：多维度数据对比
*   **箱线图 (Box Plot)**：展示数据的分布和离散程度

在选择图表类型后，你还可以自定义：

*   **Groups (X-Axis)**：横轴的分组数量，比如"一月、二月、三月"
*   **Series (Elements)**：数据系列数量，比如"销售额、成本、利润"

这样的设计让你能够灵活地适配不同的数据结构。

### 精心设计的配色方案

好的配色能让图表瞬间提升一个档次。ChartGenius 内置了 **5 套精选配色方案**：

1.  **Ocean Breeze（海洋微风）**：清新的蓝绿色系，适合科技、数据类展示
2.  **Sunset Glow（落日余晖）**：温暖的橙红色系，适合创意、活力类主题
3.  **Neon Forest（霓虹森林）**：鲜艳的绿色系，适合环保、自然类主题
4.  **Cyberpunk（赛博朋克）**：炫酷的紫粉色系，适合科幻、前卫类风格
5.  **Slate Mono（石板灰阶）**：简洁的灰色系，适合商务、正式场合

当然，如果这些配色都不满足你的需求，ChartGenius 还支持**自定义配色**。你可以根据自己的喜好或品牌色调，自由调整每个数据系列的颜色，打造独一无二的图表风格。

最重要的是，**切换配色方案是实时的**——点击一下就能看到效果，不满意就换一个，直到找到最合适的那一款。

### 灵活的数据处理

ChartGenius 提供了两种数据输入方式：

#### 1. 导入 JSON 数据

如果你已经有现成的数据，可以直接导入 `.json` 格式的文件。这对于需要频繁更新数据或者从其他系统导出数据的场景非常方便。

JSON 格式大概是这样的：

```json
[  
 { "name": "A", "series0": 70, "series1": 50 , "series2": 30},  
 { "name": "B", "series0": 80, "series1": 60 , "series2": 20},  
 { "name": "C", "series0": 40, "series1": 90 , "series2": 10},  
 { "name": "D", "series0": 60, "series1": 70 , "series2": 40},  
 { "name": "E", "series0": 50, "series1": 80 , "series2": 30}  
]
```
其中，name 为 Groups 的名字，series0~series3 为 Series 的名字，均可以自定义。

#### 2. 随机生成数据

如果你只是想快速预览某种图表的效果，或者做一个演示原型，可以使用**随机生成数据**功能。一键生成，立即可视化，非常适合快速试错和探索不同的图表类型。

### 一键导出图片

图表做好后，你可以直接**导出为图片**，然后用在 PPT、报告、论文或者社交媒体上。不需要截图，不需要担心分辨率问题，导出的图片质量足够高，可以直接使用。

ChartGenius 可以用在很多地方：

*   **课程作业和实验报告**：快速生成专业的数据图表

## 技术实现

ChartGenius 是基于 Web 技术开发的，在 gemini 的辅助下完成，主要使用了：

*   **前端框架**：构建交互式界面
*   **图表库**：实现各种图表的绘制和渲染

整个工具都在浏览器中运行，所以**所有数据都在本地处理**，不会上传到服务器，可以放心使用。
